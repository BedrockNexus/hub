# Bedrock Nexus Search Authority Plan

## Linkable Pages

- Keep every published server, approved project, public creator, and public organization on a stable canonical URL.
- Give detail pages clear share actions and link owners, members, servers, and projects to one another.
- Encourage creators to use their Bedrock Nexus URL in Discord descriptions, project documentation, GitHub repositories, YouTube descriptions, and server websites.

## Community Outreach

- Build a small partner list of Bedrock server networks, addon studios, creator communities, hosting providers, and documentation sites.
- Offer useful directory pages or creator profiles as the destination instead of requesting links to the home page.
- Publish a short listing guide that community moderators can include in resource channels without promotional copy.

## Pages Worth Linking To

- Expand category and game-version pages only when enough published content exists to make each page useful.
- Keep server status, supported versions, screenshots, reviews, version files, and changelogs current.
- Add original Bedrock ecosystem guides later only when the blog is intentionally brought out of deferred scope.

## Measurement

- Review indexed pages, search queries, click-through rate, referring domains, and broken backlinks monthly.
- Track shared links and outbound actions in first-party analytics without storing sensitive visitor data.
- Remove or redirect duplicate and dead public URLs instead of allowing thin pages to remain indexed.
